Configuration requise

Une clé API YouTube est nécessaire pour un fonctionnement optimal

📋 Comment configurer :

1. Aller sur Google Cloud Console
2. Créer un nouveau projet ou sélectionner existant
3. Activer "YouTube Data API v3" dans la bibliothèque
4. Créer une clé API dans "Identifiants"
5. Copier la clé et la coller dans le code (ligne 35)
🔑 Obtenir la clé API

💡 Pourquoi une clé API ?

• Rapidité : Détection instantanée des diffusions

• Précision : 99% de fiabilité vs 70% sans API

• Informations : Titre, durée, spectateurs

• Gratuit : 10,000 requêtes/jour gratuites
